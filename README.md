## git-alais 


```bash 
git config --global alias.uncommit 'reset --soft HEAD^'
``` 

## git remote 

```bash 
git remove -v 
``` 

add new remote : 

```bash 
git remote add exampleurl
```


git rebase feature to master : 

```bash 
git checkout feature
git rebase master 
```
